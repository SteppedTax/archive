Описание: Данная утилита написана на языке программирования Python с использованием библиотеки TKinter, затем скомпилированная в универсальный .exe файл. Оригинальный .py файл имеет 100 строк и весит только 2 килобайта (2080 байт). Все исходные файлы и дополнительные ресурсы можно посмотреть в папке src.
Использование: Перетащите файл расширения .libpr в файл программы. Вместе с программой поставляется файл README.libpr, который является .libpr версией данного документа, дабы запустить программу прямо тут.
Создание файлов .libpr: Файлы .libpr достаточно просты в создании, являясь незашифрованным текстовым документом в кодировке ANSI. Каждая новая строка обозначает новую страницу, поэтому в этой программе невозможно создать несколько новых строк на одной странице.
